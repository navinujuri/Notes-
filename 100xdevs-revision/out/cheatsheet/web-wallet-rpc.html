Creating a web based wallet, RPCs — Cheat Sheet

Cohort 2.0 · 9 lessons · quick reference

keccak-256 & Hashing (Ethereum)

Hash a string (ethers v6)
import { keccak256, toUtf8Bytes, id } from "ethers";
keccak256(toUtf8Bytes("hello")); // 0x1c8aff...
id("hello");                     // shorthand = keccak256(utf8Bytes(x))
ETH uses original Keccak-256, NOT the NIST-standardized SHA3-256 — they differ.
Event topic / function selector
id("Transfer(address,address,uint256)");     // 32-byte topic0
id("transfer(address,uint256)").slice(0, 10); // 0xa9059cbb (4-byte selector)
Selector = first 4 bytes of keccak256(canonical signature).
Derive ETH address from public key
// pubkey = uncompressed 64 bytes (drop 0x04 prefix)
// address = last 20 bytes of keccak256(pubkey)
import { computeAddress } from "ethers";
computeAddress("0x04..."); // -> 0x-checksummed address
Address is keccak256(pubKey)[12:], then EIP-55 checksummed (mixed-case).

JSON-RPC Protocol

Request envelope (spec 2.0)
{ "jsonrpc": "2.0", "id": 1, "method": "eth_blockNumber", "params": [] }
Stateless, transport-agnostic (HTTP POST or WS). id echoed back to match response.
Response envelope
{ "jsonrpc": "2.0", "id": 1, "result": "0x120fa1c" }
// on failure: { ..., "error": { "code": -32000, "message": "..." } }
Check for `error` before `result`; a 200 HTTP status can still carry an RPC error.
Raw curl call
curl https://eth.llamarpc.com -X POST \
  -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"eth_blockNumber","params":[]}'
Batch = send an array of request objects in one POST.

Units: Wei, Gwei, Lamports

ETH denominations
1 ETH  = 10^18 wei
1 gwei = 10^9  wei   // gas prices are quoted in gwei
All on-chain amounts are integers in wei — never floats.
SOL denominations
1 SOL = 10^9 lamports  // LAMPORTS_PER_SOL = 1_000_000_000
getBalance returns lamports; divide by LAMPORTS_PER_SOL for display.
Convert in ethers v6
import { formatEther, parseEther, parseUnits, formatUnits } from "ethers";
formatEther(1000000000000000000n);   // "1.0"
parseEther("1.0");                   // 1000000000000000000n
parseUnits("20", "gwei");            // 20000000000n
parse* returns bigint; keep amounts as bigint end-to-end to avoid precision loss.
Convert in Solana web3.js
import { LAMPORTS_PER_SOL } from "@solana/web3.js";
lamports / LAMPORTS_PER_SOL;         // SOL
sol * LAMPORTS_PER_SOL;              // lamports
eth_getBalance returns hex — do BigInt(hexResult) before dividing.

RPC Providers & Client Setup

Providers (the 'backend' node)
// ETH: Alchemy / Infura / QuickNode / public eth.llamarpc.com
// SOL: clusterApiUrl('devnet'|'mainnet-beta') / Helius / QuickNode
Wallet is frontend; the RPC node is a shared full node you query over HTTP/WS.
Ethers provider
import { JsonRpcProvider } from "ethers";
const provider = new JsonRpcProvider("https://mainnet.infura.io/v3/KEY");
await provider.getBlockNumber();
WebSocketProvider for subscriptions (new blocks, logs).
Solana connection
import { Connection, clusterApiUrl } from "@solana/web3.js";
const conn = new Connection(clusterApiUrl("devnet"), "confirmed");
Commitment: processed < confirmed < finalized. Use confirmed for UX, finalized for money.

Common Solana RPC Calls

getBalance
await conn.getBalance(pubkey); // lamports (number)
Raw: method 'getBalance', params ['<base58 pubkey>'].
requestAirdrop (devnet)
const sig = await conn.requestAirdrop(pubkey, 1 * LAMPORTS_PER_SOL);
await conn.confirmTransaction(sig);
getAccountInfo
await conn.getAccountInfo(pubkey); // { lamports, owner, data, executable }
getLatestBlockhash
const { blockhash, lastValidBlockHeight } = await conn.getLatestBlockhash();
Required to build a tx; blockhash expires (~150 slots).
getSignaturesForAddress
await conn.getSignaturesForAddress(pubkey, { limit: 10 }); // tx history
getParsedTransaction
await conn.getParsedTransaction(sig, { maxSupportedTransactionVersion: 0 });
sendTransaction
await conn.sendTransaction(tx, [keypair]);
Returns signature; confirm separately.

Common Ethereum RPC Calls

eth_getBalance
await provider.getBalance(addr); // bigint wei
// raw params: ['0xADDR', 'latest'] -> hex wei
Block tag: 'latest' | 'safe' | 'finalized' | 'pending' | hex block #.
eth_getTransactionCount (nonce)
await provider.getTransactionCount(addr, "pending");
Use 'pending' to include queued txs when building the next nonce.
eth_gasPrice / fees
const fee = await provider.getFeeData(); // { gasPrice, maxFeePerGas, maxPriorityFeePerGas }
eth_estimateGas
await provider.estimateGas({ to, value, data });
eth_call (read contract, no tx)
await provider.call({ to, data: id('totalSupply()').slice(0,10) });
Free, no state change, no signature.
eth_sendRawTransaction
await provider.broadcastTransaction(signedRawTxHex);
You sign locally, then push the signed raw tx.
eth_getTransactionReceipt
await provider.getTransactionReceipt(txHash); // null until mined; status 1=ok 0=revert

HD Wallet from Mnemonic (BIP39 / BIP44)

Generate mnemonic + seed
import { generateMnemonic, mnemonicToSeedSync, validateMnemonic } from "bip39";
const phrase = generateMnemonic(); // 12 words (128-bit)
const seed = mnemonicToSeedSync(phrase); // 64-byte Buffer
BIP44 path: m / 44' / coinType' / account' / change / index. ETH=60, SOL=501.
Derive Solana keypair (ed25519)
import { derivePath } from "ed25519-hd-key";
import nacl from "tweetnacl";
import { Keypair } from "@solana/web3.js";
const path = `m/44'/501'/${i}'/0'`;
const derived = derivePath(path, seed.toString("hex")).key;
const secret = nacl.sign.keyPair.fromSeed(derived).secretKey;
const kp = Keypair.fromSecretKey(secret);
Solana uses ed25519 with fully-hardened path (every level has ').
Derive ETH wallet (secp256k1)
import { Mnemonic, HDNodeWallet } from "ethers";
const m = Mnemonic.fromPhrase(phrase);
const w = HDNodeWallet.fromMnemonic(m, `m/44'/60'/0'/0/${i}`);
w.address; w.privateKey;
Increment index i for multiple accounts from ONE seed phrase.

Web Wallet Patterns

Sign & send (ETH)
import { Wallet } from "ethers";
const signer = new Wallet(privateKey, provider);
const tx = await signer.sendTransaction({ to, value: parseEther("0.01") });
await tx.wait(); // wait for receipt
sendTransaction auto-fills nonce/gas/chainId; signs locally then broadcasts.
Sign & send (SOL)
import { SystemProgram, Transaction, sendAndConfirmTransaction } from "@solana/web3.js";
const tx = new Transaction().add(SystemProgram.transfer({
  fromPubkey: kp.publicKey, toPubkey, lamports: 0.01 * LAMPORTS_PER_SOL }));
await sendAndConfirmTransaction(conn, tx, [kp]);
Show balances
const ethBal = formatEther(await provider.getBalance(addr));
const solBal = (await conn.getBalance(pub)) / LAMPORTS_PER_SOL;
Poll or subscribe; RPC data may lag one block behind.
Persist seed (client-side only)
// store encrypted mnemonic in localStorage / IndexedDB;
// NEVER send the seed or private key to any backend or RPC.
Whoever has the 12 words controls every derived account — treat as top secret.

Quick tips & gotchas

  • Keccak-256 != SHA3-256: Ethereum froze the pre-standard Keccak padding. Libraries labeled 'sha3' may give the wrong hash — verify against a known vector (empty string keccak = 0xc5d2...).
  • Ethereum eth_* RPC results are hex strings; wrap in BigInt(hex) before math. Balances/nonces overflow Number if you don't.
  • Hardened derivation uses the apostrophe (') meaning index + 2^31. Solana paths are fully hardened; ETH's last two levels (change/index) are not.
  • Nonce = number of txs sent from an account. Reuse or gaps stall txs — read with 'pending' tag and manage sequentially when batching.
  • Public RPC endpoints are rate-limited and sometimes lag; use a keyed provider (Alchemy/Infura/Helius) for production and dedicated methods for history.
  • Solana blockhash expires (~60-90s / ~150 slots). Fetch getLatestBlockhash right before signing or the tx is rejected as expired.
  • One mnemonic -> unlimited accounts by bumping the address index. Same 12 words give the SAME keys everywhere — a leak is total, irreversible loss.
  • eth_call and getAccountInfo are free reads (no signature, no gas); only sendRawTransaction / sendTransaction cost fees and change state.
