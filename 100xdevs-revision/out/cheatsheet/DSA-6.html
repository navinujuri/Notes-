C++ Part 2 — Cheat Sheet

Cohort 2.0 · 5 lessons · quick reference

Strings

std::string basics (prefer over C-strings)
#include <string>
std::string s = "hello";
s += " world";        // concat
s.size();  s.length(); // count
s.empty();             // bool
s[0]; s.at(0);         // at() bounds-checks
at() throws std::out_of_range; operator[] is UB out of bounds.
Common operations
s.substr(0, 5);            // pos,len
s.find("lo");             // idx or std::string::npos
s.replace(0, 5, "bye");
s.append("x"); s.push_back('!');
s.c_str();                // const char* (for C APIs)
s.compare(t);             // <0 ==0 >0
find returns std::string::npos (not -1) when not found.
Iterate / range-for
for (char c : s) std::cout << c;
for (char& c : s) c = toupper(c);   // & to mutate
Number <-> string conversion
std::string n = std::to_string(42);
int x   = std::stoi("42");
double d = std::stod("3.14");
long l   = std::stol("99");
stoi throws std::invalid_argument / std::out_of_range.
C-string (char array) — legacy
char buf[] = "hi";        // null-terminated: {'h','i','\0'}
#include <cstring>
strlen(buf); strcpy(d,s); strcmp(a,b);
Always leave room for the trailing '\0'; buffer overruns are UB.
Read input
std::cin >> s;              // one token (stops at whitespace)
std::getline(std::cin, s);  // whole line
Mixing >> then getline: flush stray newline with std::cin.ignore().

Pointers

Declare, address-of, dereference
int x = 10;
int* p = &x;    // p holds address of x
*p = 20;        // x is now 20
std::cout << p  // prints address
          << *p; // prints value
*p in a declaration = pointer type; *p in an expression = dereference.
Null pointer
int* p = nullptr;   // C++11, prefer over NULL/0
if (p) { /* safe */ }
Dereferencing nullptr is UB — always check before *p.
Pointer arithmetic / arrays
int a[3] = {1,2,3};
int* p = a;      // decays to &a[0]
*(p+1);          // == a[1] == 2
p++;             // next element (scaled by sizeof(int))
Array name decays to pointer; sizeof(a) != sizeof(p).
const and pointers (read right-to-left)
const int* p;    // ptr to const int (can't change *p)
int* const p;    // const ptr (can't change p)
const int* const p; // both fixed
Reference vs pointer
int& r = x;   // alias, can't be null, can't rebind
int* p = &x;  // can be null, reassignable
Prefer references for params/returns when null isn't meaningful.
Pointer to pointer / member access
int** pp = &p;
obj->method();    // (*obj).method()
ptr->field;       // deref + member

Dynamic Allocation

new / delete (single object)
int* p = new int(42);   // heap, initialized to 42
delete p;               // free
p = nullptr;            // avoid dangling
Every new needs exactly one matching delete or it leaks.
new[] / delete[] (arrays)
int* arr = new int[n];      // uninitialized
int* z   = new int[n]();    // zero-initialized
delete[] arr;               // MUST use delete[]
Mismatching new[] with delete (no brackets) is UB.
Smart pointers — prefer these (RAII)
#include <memory>
auto u = std::make_unique<int>(42);   // sole owner
auto s = std::make_shared<int>(42);   // ref-counted
// auto-freed at scope end; no manual delete
unique_ptr is move-only; use shared_ptr only when ownership is shared.
unique_ptr for arrays / release
auto a = std::make_unique<int[]>(n);
a[0] = 1;
int* raw = u.get();     // borrow, don't delete
u.reset();              // free early
Dangling pointer trap
int* p = new int(5);
delete p;
*p = 10;   // UB: use-after-free
Set to nullptr after delete; don't return address of local variable.

Function Call Arguments

Pass by value (copy)
void f(int x)  { x = 5; }   // caller unchanged
void g(std::string s);      // copies whole string!
Cheap for primitives; expensive for big objects — copies each call.
Pass by reference (mutate caller)
void inc(int& x) { x++; }
int a = 1; inc(a);   // a == 2
void swap(int& a, int& b);
No copy; caller's variable is modified directly.
Pass by const reference (read-only, no copy)
void print(const std::string& s);  // idiomatic for large objs
double sum(const std::vector<int>& v);
Default for non-trivial types you don't need to modify.
Pass by pointer
void inc(int* x) { if (x) (*x)++; }
int a = 1; inc(&a);
Use when arg is optional (can be nullptr) or for C-style APIs.
Default arguments
int pow(int b, int e = 2);
pow(3);      // 9
pow(3, 3);   // 27
Defaults go in the declaration only, not the definition. Must be trailing.
Arrays decay to pointers
void f(int a[], int n);  // a is really int*
// sizeof(a) inside f == sizeof(int*), not the array!
Pass length separately, or use std::vector / std::span (C++20).

Classes & Objects

Basic class + access specifiers
class Point {
public:
    int x, y;                 // accessible everywhere
    int getX() const;         // const = doesn't modify
private:
    int secret;               // class-only (default)
};
class members default to private; struct defaults to public.
Constructor / member init list
class Point {
    int x, y;
public:
    Point() : x(0), y(0) {}          // default
    Point(int a, int b) : x(a), y(b) {}
};
Point p(1, 2);
Point q{3, 4};   // brace init
Prefer init lists over assignment in body; required for const/reference members.
Destructor
class Buf {
    int* data;
public:
    Buf(int n) : data(new int[n]) {}
    ~Buf() { delete[] data; }   // cleanup
};
Called automatically at scope end / on delete. One per class.
Methods: definition & const
int Point::getX() const { return x; }  // out-of-line
void setX(int v) { x = v; }            // inline
const methods can be called on const objects; mark getters const.
this pointer & static members
void Point::setX(int x) { this->x = x; }  // disambiguate
static int count;      // shared across all instances
static int get() { return count; }        // no this
static members defined once outside class: int Point::count = 0;
Object on stack vs heap
Point p(1,2);              // stack, auto-destroyed
Point* h = new Point(1,2); // heap
h->getX();
delete h;                  // manual (or use unique_ptr)

Quick tips & gotchas

  • Prefer std::string over char[] and smart pointers (make_unique/make_shared) over raw new/delete — RAII frees automatically and avoids leaks.
  • Every new pairs with delete, every new[] with delete[]. Mismatch = UB. Better: don't call them directly.
  • std::string::find returns std::string::npos, not -1, when the substring is missing.
  • Pass large objects by const reference; primitives by value; use pointer/reference only when you must mutate the caller's variable.
  • After delete, set the pointer to nullptr to avoid dangling use-after-free; never dereference nullptr.
  • Mark getters and non-mutating methods const so they work on const objects; use member init lists in constructors (mandatory for const/reference members).
  • Arrays decay to pointers when passed to functions — sizeof gives the pointer size, so pass the length explicitly.
  • class defaults to private access, struct defaults to public — otherwise identical in C++.
