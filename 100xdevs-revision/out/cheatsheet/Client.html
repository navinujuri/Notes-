Client side ETH — Cheat Sheet

Cohort 3.0 · 7 lessons · quick reference

Setup & Providers

Install (ethers v6)
npm install ethers   # v6 (native bigint)
Browser: wrap MetaMask (injected EIP-1193 provider)
// MetaMask injects window.ethereum
const provider = new ethers.BrowserProvider(window.ethereum);
v5: new ethers.providers.Web3Provider(window.ethereum)
Connect wallet + get signer (opens MetaMask popup)
await provider.send("eth_requestAccounts", []);
const signer = await provider.getSigner();
const addr   = await signer.getAddress();
getSigner() is async in v6, was sync in v5
Read-only RPC provider (no wallet needed)
const provider = new ethers.JsonRpcProvider(
  "https://eth-sepolia.g.alchemy.com/v2/<KEY>"
);
Use for reads/server. Alchemy/Infura give you the RPC URL.
Load ethers in a plain HTML page (CDN, ESM)
<script type="module">
  import { ethers } from "https://cdn.ethers.org/lib/ethers.min.js";
</script>

ETH JSON-RPC (what everything sits on)

Every node speaks JSON-RPC over HTTP/WS — ethers is a wrapper
curl <RPC_URL> -X POST -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","method":"eth_blockNumber","params":[],"id":1}'
Core node methods
eth_blockNumber        // latest block
eth_getBalance         // wei of an address
eth_call               // read-only exec, no state change (free)
eth_sendRawTransaction // broadcast signed tx (costs gas)
eth_getTransactionReceipt
eth_chainId  eth_gasPrice  eth_estimateGas
Fire a raw RPC method through ethers
const hexBn = await provider.send("eth_blockNumber", []);
const bn = parseInt(hexBn, 16);
Wallet-only methods (only exist via MetaMask, not raw nodes)
eth_requestAccounts        // connect
wallet_switchEthereumChain // change network
wallet_addEthereumChain
personal_sign              // sign a message

Reading chain data & units

ETH balance of an address
const wei = await provider.getBalance("0xAbc...");
console.log(ethers.formatEther(wei)); // "1.5"
Block number, nonce, fees
await provider.getBlockNumber();
await provider.getTransactionCount(addr); // nonce
const f = await provider.getFeeData();    // gasPrice, maxFeePerGas
Unit conversion (wei <-> human)
ethers.parseEther("0.1");     // 100000000000000000n (bigint)
ethers.formatEther(wei);      // "0.1"
ethers.parseUnits("100", 6);  // USDC has 6 decimals
ethers.formatUnits(raw, dec);
v6 returns native bigint (n suffix), not v5 BigNumber. Chain amounts are ALWAYS wei.

Interacting with a contract (read-only)

You need: contract address + ABI (a subset is fine)
// human-readable ABI — only the fns you call
const abi = [
  "function balanceOf(address) view returns (uint256)",
  "function decimals() view returns (uint8)",
  "function transfer(address to, uint256 amt) returns (bool)"
];
Instantiate with a PROVIDER for reads
const c = new ethers.Contract(address, abi, provider);
Call view/pure functions (free, instant, no signer)
const raw = await c.balanceOf(addr);
const dec = await c.decimals();
console.log(ethers.formatUnits(raw, dec));
view/pure -> resolves to eth_call. No gas, no wallet popup.

Writing to a contract (state change)

Re-instantiate with a SIGNER (needs wallet + gas)
const signer = await provider.getSigner();
const c = new ethers.Contract(address, abi, signer);
// or reuse: c.connect(signer)
Send tx and wait for confirmation
const tx = await c.transfer(to, ethers.parseUnits("10", 6));
const receipt = await tx.wait();   // wait 1 confirmation
console.log(receipt.hash, receipt.status); // status 1 = success
await c.foo() only broadcasts; state isn't changed until tx.wait() resolves.
Overrides: send ETH along / cap gas
await c.deposit({ value: ethers.parseEther("0.5") });
await c.foo(arg, { gasLimit: 100000n });
Dry-run to catch reverts before spending gas
await c.transfer.estimateGas(to, amt); // will it fit / revert?
await c.transfer.staticCall(to, amt);  // simulate, surfaces revert
Plain ETH transfer (no contract)
const tx = await signer.sendTransaction({
  to, value: ethers.parseEther("0.01")
});
await tx.wait();

Interacting from the server (Node.js)

No MetaMask on a server — YOU hold the key
import { ethers } from "ethers";
const provider = new ethers.JsonRpcProvider(process.env.RPC_URL);
const wallet = new ethers.Wallet(process.env.PRIVATE_KEY, provider);
Never hardcode/commit keys. Load from env / secret manager.
Same Contract API — the wallet IS the signer
const c = new ethers.Contract(addr, abi, wallet);
const tx = await c.transfer(to, amt);
await tx.wait();
Create / import a wallet
const w = ethers.Wallet.createRandom(); // w.address, w.privateKey, w.mnemonic.phrase
ethers.Wallet.fromPhrase(mnemonic);     // from seed phrase

Network, events & config

Detect / assert network
const net = await provider.getNetwork();
net.chainId; // 1n mainnet, 11155111n sepolia, 137n polygon
Ask MetaMask to switch chains (chainId is HEX here)
await window.ethereum.request({
  method: "wallet_switchEthereumChain",
  params: [{ chainId: "0xaa36a7" }] // 11155111 sepolia
});
ethers uses decimal chainId; wallet_* RPC params use hex.
React to wallet changes
window.ethereum.on("accountsChanged", (a) => setAccount(a[0]));
window.ethereum.on("chainChanged", () => window.location.reload());
Contract events (live + historical)
c.on("Transfer", (from, to, amt, ev) => { /* live */ });
const logs = await c.queryFilter("Transfer", fromBlock, toBlock);

Quick tips & gotchas

  • Reads vs writes: view/pure = eth_call (free, instant, provider is enough); state changes = signed tx (gas + wallet + tx.wait()).
  • ethers v6 uses native bigint everywhere; v5 used BigNumber. Don't mix versions — check which one the project pins.
  • Always await tx.wait() before assuming state changed. receipt.status === 1 = success; a revert throws.
  • Key custody rule: browser -> MetaMask signs (you never see the key); server -> ethers.Wallet + private key from env. Never ship a key to the client.
  • ABIs can be a subset — include only the functions you actually call. Human-readable ABI beats pasting giant JSON.
  • getSigner() is async in v6 (returns a Promise); it was synchronous in v5 — a common migration bug.
  • Guard for missing wallet: if (!window.ethereum) prompt to install MetaMask before calling any eth_* method.
  • chainId is decimal in ethers (11155111n) but hex in wallet_* RPC params ("0xaa36a7"). Use staticCall/estimateGas to catch reverts before spending gas.
